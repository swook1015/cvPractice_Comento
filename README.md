#1주차 전처리 과정 설명
1. datasets을 통해 전처리에 필요한 데이터를 실시간(streaming)으로 가져옴.
2. 크기 조정
   가져온 사진을 224x224 사이즈로 바꿈
3. 데이터 증강
   좌우반전, -15~15도 사이를 랜덤하게 회전, 색상 변환을 적용
4. 노이즈 제거
   GaussianBlur를 이용하여 지저분한 점들을 제거
5. 데이터 값이 0-255이면, 경사 하강법을 적용했을 때 오랜 시간이 걸린다.
   따라서, 0-1사이의 값으로 정규화 하여 최적의 정답을 더욱 빠르게 도달할 수 있게 적용
6. 전처리 된 이미지를 "preprocessed_samples" 파일에 저장.

#2주차 과정 설명
[구조 설계]
1. Load Step: data 폴더 파일 안에 있는 전체 이미지 파일 읽기
2. Process Step: 흑백 변환 및 Depth Map 생성 (핵심 로직)
3. Transform Step: X, Y, Z 좌표를 결합하여 3D 포인트 클라우드 생성
4. Save Step: 결과 이미지 및 .npy 파일 저장
